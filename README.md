# 黑龙江工程学院教务处助手
##目前实现的功能：
- 课程表导出.ics文件，可以导入google日历等应用软件
- 更多功能敬请期待

##环境要求
1. Python3.6+
2. BeautifulSoup4 4.9.1+
3. PIL 7.0.0+
4. requests 2.22.0+
